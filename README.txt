Description:

An event board and social platform for billiards players to connect with each other; find, upload, and register for events

Features:
- Login & Registration
- Dashboard with list of all events and events player has registered for
- View event details
- Add events
- Edit events
- Create/edit player profile
- View player profiles

Feature backlog:
- Change account settings
- Add friends/view friends list
- Team pages (create, edit, join)
- Registration fee w/ Stripe API
